# Python Combination Calculator

## Description
This project is a simple Python program that calculates the combination value C(n, p) (also called binomial coefficient).

The formula used is:

C(n, p) = n! / (p! × (n-p)!)

## Features
- Input validation for p (1 ≤ p ≤ n)
- Factorial calculation function
- Combination calculation function
- Simple command-line interface

## How It Works

The program contains three main functions:

- `saisir()`  
  Takes the values of `n` and `p` from the user and checks that `p` is valid.

- `fact(x)`  
  Calculates the factorial of a number.

- `calculer(n, p)`  
  Computes the combination C(n, p).
